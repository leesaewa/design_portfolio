가스온 관리자 페이지

가스온이라는 클라이언트의 관리자 페이지 디자인을 맡았습니다. 가스온은 LPG와 가스 난방용기 충전 사업을 주로 하는 클라이언트입니다. 충전 사업을 주로 하다보니, 소비자가 좀 더 쉽게 관리할 수 있도록 관리자 페이지에 로그인해서 들어가면 머니를 충전할 수 있고, 충전된 머니를 사용하여 다시 가스를 충전할 수 있도록 하는 페이지였습니다.

관리자 페이지 디자인은 처음이었고, 서브페이지 전체를 디자인하고 코딩해야 해서 조금 버거웠던 기억이 있습니다. 그리고, 일반 사이트와 달리 프로그래밍이 들어가는 사이트여서 개발자가 알아보기 쉽게 디자인/코딩하기 위해 노력했습니다. 아쉽게도 모바일은 없고 PC만 구현했습니다.
